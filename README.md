Strands-Agent

Strands-Agent is a beginner-friendly AI agent built using the Strands framework and Ollama LLM.
It can answer questions in a kind and humble way and use tools like HTTP requests.

✅ Features

* Uses OllamaModel to process user queries.

* Includes a system prompt to guide agent behavior.

* Can use tools like http_request to fetch information from free APIs.

* Easy to run and understand for beginners.


 ? What You Need

* Python 3.9 or higher

* Strands library

* Strands-tools library

* Ollama LLM running locally at http://localhost:11434/

⚡ Step-by-Step Setup

1) Clone the project

git clone https://github.com/gopeshthakare13/devops-2026-projects.git

cd devops-2026-projects

2) Create a virtual environment (recommended)

python -m venv venv

3) Activate the virtual environment

venv\Scripts\activate    # Windows:

4) Install dependencies

pip install strands

pip install strands-tools
